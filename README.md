# Northframe Showcase

Single-page React + Vite site pentru un studio de videografie orientat spre brand/comercial.

## Cerinte

- Node.js 20+
- npm 10+

## Instalare

```bash
npm install
npm run dev
```

## Variabile de mediu

Copiaza `.env.example` in `.env` si completeaza:

- `VITE_EMAIL_SERVICE_ID`
- `VITE_EMAIL_TEMPLATE_ID`
- `VITE_EMAIL_PUBLIC_KEY`

Formularul foloseste endpoint-ul EmailJS direct din frontend.

## Media placeholders

Poti inlocui rapid asset-urile demo din `public/media`.

- Hero video: `public/media/hero-showreel.mp4`
- Hero poster: `public/media/hero-poster.svg`
- Proiecte: `public/media/project-*.svg`
